# Volunteer Management Application - Oracle APEX

This project is a **volunteer and event management system** developed to help organizations coordinate volunteers, activities, and participation records efficiently.

The application was created as part of an **undergraduate thesis project at Universidad Americana (Paraguay)**. The objective of the project was to design and implement a digital platform that supports community and non-profit organizations in managing their volunteer programs.

## Project Overview

The system allows organizations to:

- Register and manage volunteers
- Create and organize events
- Track volunteer participation
- Maintain structured records of community activities
- Generate reports to monitor volunteer engagement

This project demonstrates how **Oracle APEX (Application Express)** can be used to rapidly develop **data-driven web applications** that support real-world organizational needs.

## Technologies Used

- **Oracle APEX (Application Express)**
- **Oracle Database**
- **SQL and PL/SQL**
- Web technologies used within APEX (HTML, CSS, JavaScript)

## Purpose of the Project

The main goal of this application is to show how **Oracle technologies can support volunteer coordination and community initiatives** through efficient data management and web-based interfaces.

The system was designed to provide a practical solution for organizations that rely on volunteer participation and need a structured way to manage events and activities.
